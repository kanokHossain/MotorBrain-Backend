# MotorBrain Product Roadmap

## Phase 1 — MVP (Current)

Core functionality for individual vehicle owners.

- [x] User registration and JWT authentication
- [x] Multi-vehicle support (car and bike)
- [x] Vehicle maintenance logging
- [x] Maintenance reminders (time + mileage based)
- [x] Mileage and odometer tracking
- [x] Fuel refill logging with efficiency calculation
- [x] Push notifications via Firebase FCM
- [x] Basic subscription model (Free/Basic/Premium)
- [x] In-app notification center
- [x] REST API with Swagger docs
- [x] React Native mobile app (iOS and Android)

---

## Phase 2 — AI & Analytics

Intelligence layer on top of the core data.

- [ ] AI vehicle health scoring (rule-based v1)
- [ ] AI predictive maintenance (LLM-powered v2 via Claude API)
- [ ] Fuel efficiency trend charts
- [ ] Maintenance cost analytics
- [ ] Annual expense reports (PDF export)
- [ ] Smart reminder suggestions after maintenance logging
- [ ] Vehicle comparison (if user has multiple vehicles)

---

## Phase 3 — Platform Expansion

Beyond the mobile app.

- [ ] Web dashboard (React or Next.js)
- [ ] Service history sharing (PDF or link — useful for resale)
- [ ] Service center integration (book appointments from app)
- [ ] OBD-II device integration (real-time diagnostics)
- [ ] Multi-language support

---

## Phase 4 — Fleet Management

Expansion for small businesses and fleets.

- [ ] Fleet owner and driver roles
- [ ] Fleet dashboard with multi-vehicle overview
- [ ] Driver assignment to vehicles
- [ ] Fleet cost analytics and reporting
- [ ] Bulk import of vehicle and maintenance data

---

## Future Ideas

- Marketplace for spare parts (affiliate)
- Community forum and vehicle tips
- Insurance integration
- Resale value estimator based on maintenance history
- Warranty tracking
